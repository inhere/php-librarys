# asset manager

## AssetLoad

this is a asset load tool.
class is at `inhere\librarys\asset\AssetLoad`

usage:

```
<?php
use inhere\librarys\asset\AssetLoad;

AssetLoad::loadCss([
    'xx/ss.css',
    'xx/zz.css',
]);
?>

<?php 
AssetLoad::loadJs([
    'xx/ss.js',
    'xx/zz.js',
]);

?>
```
